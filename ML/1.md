🎯 회귀(Regression)란?  
숫자를 예측하는 방법다.  
  
🧒 예시:  
📘 상황: 내가 아이스크림 가게를 운영한다고 상상해보자!
하루에 기온이 얼마나 되는지에 따라 아이스크림이 얼마나 팔리는지 기록해봤다.  
| 기온 (℃) | 아이스크림 판매 개수 |
| ------ | ----------- |
| 20도    | 100개        |
| 25도    | 150개        |
| 30도    | 200개        |
  
이제 나는 궁금해졌다.  
“음... 오늘 27도면 아이스크림 몇 개 팔릴까? 🤔”  
바로 이럴 때 회귀를 쓴다!  
  
  
🧠 회귀는 어떻게 도와주냐면:  
1. 너가 모아놓은 데이터를 보고
2. 숫자 사이의 관계를 배우고
3. 새로운 숫자가 들어오면 (예: 27도)
4. 거기에 맞는 예상 숫자를 알준다! (예: 아이스크림 180개쯤 팔릴 거야~)
  
```
기온이 높아질수록 → 아이스크림 판매량도 높아짐
회귀는 그 관계를 선으로 그려서
그 선을 따라 새로운 값을 예측해주는 거야!
```  
  
💡 정리!
**분류(Classification)**는 → “이건 고양이야? 강아지야?” 처럼 종류를 맞추는 것
**회귀(Regression)**는 → “얼마야? 몇 개야?” 처럼 숫자를 맞추는 것  
  
---
**선형회귀**  
**☑️ 선형 회귀**
- 선형회귀는 종속 변수와 하나 이상의 독립 변수 간의 선형 관계를 모델링 하는 방법입니다
- 독립변수의 수에 따라 단순 선형회귀와 다중 선형회귀로 나뉩니다

독립변수?: 우리가 넣어주는 값 (문제)  
종속변수?: 모델이 예측하려는 값 (정답)  
예시:  
| 날씨 (℃) | 아이스크림 판매량 |
| ------ | --------- |
| 20도    | 100개      |
| 25도    | 150개      |
| 30도    | 200개      |
  
날씨 (℃) → 우리가 아는 값 → 독립변수  
아이스크림 판매량 → 예측하고 싶은 값 → 종속변수  
  
📦 용어 정리  
| 용어       | 뜻                                  | 쉽게 말하면 |
| -------- | ---------------------------------- | ------ |
| **독립변수** | 결과에 영향을 주는 변수 (우리가 넣는 값)           | 문제     |
| **종속변수** | 독립변수에 따라 값이 결정되는 변수 (우리가 맞히고 싶은 값) | 정답     |
  
  ✅ 선형 회귀에서는?
독립변수(X)를 가지고
종속변수(y)를 예측하는 게 목표다!
즉,  
X → y를 **직선(선형)**으로 예측하는 게 선형 회귀다!    
  
☑️ 단순 선형 회귀 VS 다중 선형 회귀
