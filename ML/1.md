🎯 회귀(Regression)란?  
숫자를 예측하는 방법다.  
  
🧒 예시:  
📘 상황: 내가 아이스크림 가게를 운영한다고 상상해보자!
하루에 기온이 얼마나 되는지에 따라 아이스크림이 얼마나 팔리는지 기록해봤다.  
| 기온 (℃) | 아이스크림 판매 개수 |
| ------ | ----------- |
| 20도    | 100개        |
| 25도    | 150개        |
| 30도    | 200개        |
  
이제 나는 궁금해졌다.  
“음... 오늘 27도면 아이스크림 몇 개 팔릴까? 🤔”  
바로 이럴 때 회귀를 쓴다!  
  
  
🧠 회귀는 어떻게 도와주냐면:  
1. 너가 모아놓은 데이터를 보고
2. 숫자 사이의 관계를 배우고
3. 새로운 숫자가 들어오면 (예: 27도)
4. 거기에 맞는 예상 숫자를 알준다! (예: 아이스크림 180개쯤 팔릴 거야~)
  
```
기온이 높아질수록 → 아이스크림 판매량도 높아짐
회귀는 그 관계를 선으로 그려서
그 선을 따라 새로운 값을 예측해주는 거야!
```  
  
💡 정리!
**분류(Classification)**는 → “이건 고양이야? 강아지야?” 처럼 종류를 맞추는 것
**회귀(Regression)**는 → “얼마야? 몇 개야?” 처럼 숫자를 맞추는 것  
  
---
**☑️ 선형 회귀**
- 선형회귀는 종속 변수와 하나 이상의 독립 변수 간의 선형 관계를 모델링 하는 방법입니다
- 독립변수의 수에 따라 단순 선형회귀와 다중 선형회귀로 나뉩니다

독립변수?: 우리가 넣어주는 값 (문제)  
종속변수?: 모델이 예측하려는 값 (정답)  
예시:  
| 날씨 (℃) | 아이스크림 판매량 |
| ------ | --------- |
| 20도    | 100개      |
| 25도    | 150개      |
| 30도    | 200개      |
  
날씨 (℃) → 우리가 아는 값 → 독립변수  
아이스크림 판매량 → 예측하고 싶은 값 → 종속변수  
  
📦 용어 정리  
| 용어       | 뜻                                  | 쉽게 말하면 |
| -------- | ---------------------------------- | ------ |
| **독립변수** | 결과에 영향을 주는 변수 (우리가 넣는 값)           | 문제     |
| **종속변수** | 독립변수에 따라 값이 결정되는 변수 (우리가 맞히고 싶은 값) | 정답     |
  
  ✅ 선형 회귀에서는?
독립변수(X)를 가지고
종속변수(y)를 예측하는 게 목표다!
즉,  
X → y를 **직선(선형)**으로 예측하는 게 선형 회귀다!    
  
### ☑️ 단순 선형 회귀 VS 다중 선형 회귀  
🎯 먼저 한 줄 요약!  
종류	뜻
단순 선형 회귀: 독립변수가 1개인 경우.  
다중 선형 회귀: 독립변수가 2개 이상인 경우. 
  
#### 📘 단순 선형 회귀: 하나의 원인으로 결과를 예측 
하나의 독립 변수(X)가 종속 변수(y)에 영향을 준다고 가정하는 모델이다.  
🍦 예시:  
| **기온(℃)** | **아이스크림 판매량** |
| --------- | ------------- |
| 20도       | 100개          |
| 25도       | 150개          |
| 30도       | 200개          |
  
- 여기서 기온(℃)이 하나의 독립변수예요.  
- 아이스크림 판매량이 종속변수예요.  
- 관계: 기온이 오를수록 → 아이스크림이 더 팔린다  
  
🎓 비유: “기온이 올라가면 아이스크림을 더 먹고 싶어져!”  
이렇게 기온 하나만 가지고 판매량을 예측하는 게 단순 선형 회귀예요.  
  
#### 📘 다중 선형 회귀: 여러 원인으로 결과를 예측
여러 개의 독립 변수들이 종속 변수에 영향을 준다고 가정하는 모델이다.  
  
🍦 예시:  
| **기온(℃)** | **습도(%)** | **아이스크림 판매량** |
| --------- | --------- | ------------- |
| 25도       | 60%       | 150개          |
| 30도       | 80%       | 170개          |
| 35도       | 90%       | 200개          |
  
- 기온 + 습도 → 두 개의 독립변수
- 아이스크림 판매량 → 종속변수  
  
🎓 비유: 기온이 높고 습도까지 높으면 너무 덥고 끈적하니까 아이스크림을 더 사먹는다!”
이렇게 **여러 가지 조건(기온 + 습도)**을 고려해서 예측하는 게 다중 선형 회귀예요.   
  
⚖️ 표로 정리:  
| 구분       | 독립 변수 수 | 예시             | 설명              |
| -------- | ------- | -------------- | --------------- |
| 단순 선형 회귀 | 1개      | 기온 → 아이스크림 판매량 | 하나의 원인으로 결과를 예측 |
| 다중 선형 회귀 | 2개 이상   | 기온 + 습도 → 판매량  | 여러 원인으로 결과를 예측  |
  
✅ 결론: 
- 단순 선형 회귀는 한 가지 정보로 예측
- 다중 선형 회귀는 여러 정보를 함께 고려해서 예측  
  
### 🧠 Scikit-learn이 뭐야?
**Scikit-learn(사이킷런)**은
**파이썬으로 머신러닝을 쉽게 할 수 있게 도와주는 도구(라이브러리)**이다.  
  
🎓 쉽게 말하면?  
머신러닝을 하려면 사실 수학 계산이 엄청 많다… 😵  
근데 사이킷런을 쓰면 그걸 한 줄 코드로 간단히 할 수 있다!  
  
📦 예를 들어:  
- 선형 회귀
- 로지스틱 회귀
- K-최근접 이웃(KNN)
- SVM
- 결정 트리
- 모델 평가, 분할, 전처리 등…  

➡️ 이런 걸 다 구현해주는 똑똑한 도구 상자다!  
  
🧰 비유하자면?  
사이킷런은 레고 블록 같은 거다!  
우리가 머신러닝 모델을 직접 수학으로 다 만들 필요 없이,  
블록을 조립하듯 모델을 쉽게 만들 수 있다!  
  
```
from sklearn.model_selection import train_test_split   # 데이터 분할
from sklearn.linear_model import LinearRegression      # 선형 회귀 모델
from sklearn.metrics import mean_squared_error, r2_score  # 평가 지표
```
  
✅ 정리하면. 
| 이름               | 설명                                               |
| ---------------- | ------------------------------------------------ |
| **Scikit-learn** | 머신러닝용 파이썬 라이브러리 (아주 유명하고 강력해요!)                  |
| **우리가 한 것**      | Scikit-learn으로 선형 회귀 모델을 만들고, 학습하고, 예측하고, 평가했어요! |
